CLI Tool to detect website using Cloudflare

# Installation
```sh
$ git clone https://github.com/HanzHaxors/CloudflareDetector.git
$ cd CloudflareDetector
$ pip3 install -r requirements.txt
$ python3 CloudflareDetector.py <protocol://><URL> [-v]
```
